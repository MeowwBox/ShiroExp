## 0x02 说明

里面很多payload都是各位大佬的研究成果。  
写这样一个工具主要是为了学习漏洞原理后，输出对应的内容或工具。

该轮子主要功能如下：
1. 支持多条cc利用链
2. 多中间件通用回显
3. 内存马注入
4. 默认秘钥爆破
